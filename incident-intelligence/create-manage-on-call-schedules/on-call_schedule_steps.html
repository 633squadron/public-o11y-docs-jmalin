1. In Incident Intelligence, go to Incident Management.
  2. Select the On-call schedules tab.
  3. Select Create new schedule
  4. Select the pencil icon to edit the schedule name. Name your schedules so that they are easy to find when adding them to an incident work. For example, schedule names should include a team name, a priority (for example, primary, secondary, and so on), and what is being covered (for example, database, web, and so on). Select the checkmark icon to save your schedule name.
  5. (Optional) Add a teams tag if you want to associate your schedule with a team. This makes finding a schedule in an incident workflow easier. To add a teams tag, enter the team name in the Teams tag field and select the team when it appears.
  6. Under Add shifts to this schedule, create one day-by-day shift and one week-by-week shift. There is a default blank shift to get you started in new schedules. To add shifts to the schedule, select Add new shift.
    1. Enter a shift name. Shift names often refer to time (for example, business hours, weekends, and so on).
    2. Select a shift start date.
    3. Use the timezone picker to select the timezone for the coverage.

    4. Define your shift coverage:

      Shift coverage type

      Steps

      Day-by-day shift coverage
      1. Select the days of the weeks you want to set up coverage for.
      2. Select the Daily start time and Daily end time. If you want to set up 24-hour coverage, select Create 24 hr shifts.

      Week-by-week shift coverage.
      1. Select the day and time you want the shift to start in the Starts on drop-down lists.
      2. Select the day and time you want the shift to end in the Ends on drop-down lists.
    5. Under Customize shift handoff, select the number of shifts that you want to elapse before a handoff to the next team member in the schedule list.
    6. Under Add responders, add one or more team members to the shift.
      1. Enter team member names and select them when they appear. If you added a teams tag in step 5, the team members from this team are listed at the top of this drop-down menu list. Select Add all members from [TEAM] to quickly add all the team members from that team. The ALL section of the team member list comes from the Observability Cloud user list. See Create and manage users in Splunk Observability Cloud for more info.
      2. Drag responders in the list to reorder.
    7. Select Save shift. Shifts are added to the SAVED SHIFTS list under the schedule name.
